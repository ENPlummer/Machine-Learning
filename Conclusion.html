Conclusion

Since The Support Vector Machine and the Logistic Regression models produced scores of 1.0 both models are out of consideration for the best performing model. A score of 1.0 means that the model is working pefectly which is highly unlikely and is suspect. The 1.0 scores occured most liekly due to an error in creating the models.

With the Support Vector Machine and Logistic Regression models being out of consideration, the best performing model was the deep learning neural network model with a score of 0.9633. The worst perfoming model was the normal neural network model with an accuracy rate of 0.9083.

In is very important that your data fits the model. The classification models don't work with continuous data set. A categorical data is needed for classfication models. For continous data sets the regression version of the models have to be used.

Issues

A major issue was that the Support Vector Machine and the Logistic Regression models produced accuracy rates of 100% which indicates that errors were made in creating the model. A score of 1.0 accuracy are always suspect.

The data set consisted of categorical data and preprocessing had to be done in order to be able to use the models.
