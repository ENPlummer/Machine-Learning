Machine Learning Models

Logistic Regression

Logistic regression is a supervised learning algorithm that is used to predict a binary set of classes or categories. For example: yes/no, young/old, or happy/sad.

Random Forest

The basis of the random forest algoritm is the decison tree. A decision tree encodes a series on true/false questions. The problem with the basic decision tree is that it doesn't handle complex data well. Decision trees can become very complex and not generalize well. The random forest algorithm solves this issue by sampling the data instead of producing one complex tree, random forest produces severla simpler decision trees. Each tree is built from a subset of data. While a single tree is considered a "weak classifier", but when you combine the trees together they become a "strong classifier".

Support Vector Machine

Is a classification algoritm that draws the optimal line between two categories

KNN Nearest Neighbor

Is a robust algorithm used for classification and sometimes for regression. The goal of KNN is to predict data points based on known data points in the feature space.

Neural Networks

Neural networks are connected neurons very much like the human brain. Neural networks processes information very much like the human brain processes visual information. The most important mathmatical detail of neural networks is that neural networks output probablities.
